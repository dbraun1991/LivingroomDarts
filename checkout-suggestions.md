## Single Out

Last dart can be any segment (single, double, triple, bull). No double required.

**Still impossible in 3 darts:** 163, 165, 166, 169 (no valid 3-segment combination exists)

**Newly possible vs. Double Out:**

| Score | D1  | D2  | D3  |
|-------|-----|-----|-----|
| 168 | T20 | T20 | T16 |
| 162 | T20 | T18 | T16 |
| 159 | T20 | T19 | T14 |

All other scores 101–170 use the same routes as Double Out.

## Implementation Notes

- **Threshold for showing suggestions:** ≤ 170 for both rules
- **dartsLeft:** 3 minus darts already thrown this visit — only show suggestions that fit within remaining darts
- **Impossible scores:** return `null` — show nothing, not a placeholder
- **Single out ≤ 60:** prefer 1-dart finish (triple/single) over 2-dart double-out path
- **Score 99 (double out):** requires 3 darts — T19 S10 D16 (fits 3-dart slot, show all three)
